# 🥠 Digital Fortune Cookie  
**“행운을 기다리지 마세요. 와서 가져가세요.”**  

## 프로젝트 소개

> 운세를 랜덤으로 제공하는 **디지털 포춘 쿠키** 웹 애플리케이션입니다.<br />
> 사용자는 하루 한 번 포춘 쿠키를 열 수 있고,<br />
> 지난 7일 간의 행운 점수를 확인하거나 운세 랭킹을 확인할 수 있습니다.
- 배포 URL : http://cookie4u.store/
